카카오톡/위챗 백업 시작

처음 볼 것은 두 개뿐입니다.
- 안내를 보려면 이 파일을 봅니다.
- 바로 시작하려면 1_백업_시작.bat를 더블클릭합니다.
  Windows가 파일 확장자를 숨기면 1_백업_시작으로 보일 수 있습니다.

나중에 정리할 때만 5_정리_삭제.bat를 사용합니다.
4_준비_보고서.bat와 4_검증_보고서.bat는 지원용 보고서를 만들 때만 사용합니다. 처음 백업 시작 파일이 아닙니다.
백업 전 준비 상태만 보내야 하면 첫 화면이나 준비 확인의 준비 보고서를 누릅니다.
브라우저를 열기 어려운 상태에서 준비 상태만 보내야 하면 4_준비_보고서.bat를 실행합니다.
백업 뒤 실사용 검증 보고서가 필요하면 첫 화면의 막힐 때만 자세히 보기 안에 있는 위챗 찾기 / 막힐 때 영역에서 검증 보고서를 누르거나, 진행 기록에서 검증 보고서를 누릅니다.
보고서에 "요청 시간이 초과됐습니다"가 보이면 브라우저와 검은 창을 닫고 1_백업_시작.bat를 다시 실행합니다.
브라우저를 찾기 어려울 때만 4_검증_보고서.bat를 실행합니다.
막힐 때만 3_문제_확인.txt를 엽니다.
다른 파일과 폴더는 자동 실행과 참고 문서입니다. 처음 사용하는 사람은 열 필요가 없습니다.
첫 카카오톡/위챗 백업은 Windows 앱과 문자 인식만 준비하면 시작할 수 있습니다. 예전 백업 파일 검사나 추가 검수 같은 특별 기능은 처음에는 건너뛰어도 됩니다.

1. 이 폴더에서 1_백업_시작.bat를 더블클릭합니다.
   Windows가 파일 확장자를 숨기면 1_백업_시작으로 보일 수 있습니다. 보이지 않으면 시작하기 또는 시작하기.bat를 더블클릭합니다.
   ZIP 파일 안에서 바로 실행하지 않습니다. 먼저 압축을 풀고, scripts와 web 폴더가 같은 폴더에 보이는 상태에서 실행합니다.
   "필수 실행 파일이 빠졌습니다"가 보이면 압축이 제대로 풀리지 않았거나 일부 파일만 옮긴 상태입니다. 폴더 전체를 다시 압축 해제한 뒤 1_백업_시작.bat를 실행합니다.
   "백업 저장 폴더를 준비하지 못했습니다" 또는 저장할 수 없다는 안내가 나오면 압축을 푼 폴더를 바탕화면이나 문서처럼 내가 저장할 수 있는 일반 폴더로 옮긴 뒤 1_백업_시작.bat를 다시 실행합니다.
   OneDrive 동기화 오류가 난 폴더, 회사 보안 폴더, 읽기 전용 폴더는 피합니다.
   Windows가 실행 확인 창을 띄우면 GitHub Releases에서 받은 공개 ZIP인지 확인한 뒤 실행 또는 허용을 누릅니다.
   모르는 출처에서 받은 파일이면 실행하지 않습니다.
   백업 화면 실행 도구 설치가 막혔다는 안내가 나오면 자동으로 만들어지는 4_실행도구_설치.url을 더블클릭하고, 설치가 끝난 뒤 1_백업_시작.bat를 다시 실행합니다.
   브라우저가 자동으로 열리지 않으면 새로 만들어진 2_백업_화면.url을 더블클릭합니다. 위챗만 바로 열려면 2_1_위챗_백업.url, 카카오톡만 바로 열려면 2_2_카카오톡_백업.url을 더블클릭합니다. 여러 방을 한 번에 저장하려면 2_3_위챗_통째백업.url 또는 2_4_카카오톡_통째백업.url을 더블클릭합니다.

2. 브라우저가 열리면 첫 화면에서 위챗 백업, 카카오톡 백업, 결과 보기 중 필요한 작업을 고릅니다.
   브라우저가 화면을 준비하는 중에도 같은 첫 화면에 여러 방을 한 번에 저장, 내 컴퓨터에 저장, 외부 전송 기본 꺼짐, 입력할 내용 없음 안내가 먼저 보입니다.
   처음에는 위챗 백업 또는 카카오톡 백업을 누릅니다.
   여러 방을 한 번에 저장하려면 같은 첫 화면의 위챗 통째 백업 확인 또는 카카오톡 통째 백업 확인을 누릅니다.
   앱 상태나 문자 인식 확인은 막힐 때만 자세히 보기 안에 있습니다.
   위챗이나 카카오톡이 설치되어 있지 않으면 화면의 공식 설치 페이지 버튼을 누르고 설치와 로그인을 마친 뒤 상태 새로고침을 누릅니다.

3. 위챗 또는 카카오톡 앱에서 백업할 방을 선택합니다.
   브라우저로 돌아와 방 선택 완료, 앱 창 앞에 둠 표시를 선택하고 백업을 누릅니다.
   방 선택 완료, 앱 창 앞에 둠 표시는 체크박스입니다. 앱에서 방을 선택하고 앱 창을 앞에 둔 상태인지 확인한 뒤 체크하고 백업을 누릅니다.
   체크를 빼먹고 백업을 눌러도 "방 선택 완료 표시가 필요합니다" 카드가 나오며, 그 카드의 방 선택 완료 표시로 이동을 누르면 체크 위치로 돌아가고 잠깐 강조됩니다.
   백업이 시작되면 끝날 때까지 위챗 또는 카카오톡 창을 최소화하거나 다른 창으로 가리지 않습니다.
   카카오톡이나 위챗에서 여러 방을 한 번에 저장하려면 첫 화면의 위챗 통째 백업 확인 또는 카카오톡 통째 백업 확인을 누릅니다.
   이 흐름이 백업 화면의 여러 방 저장 영역입니다.
   누른 뒤에는 앱 창을 앞에 두고 왼쪽 채팅 목록이 보이게 둡니다.
   준비가 끝난 상태면 백업 화면의 전체 목록 확인 준비 완료 체크 위치로 이동합니다.
   앱 창을 앞에 두고 왼쪽 채팅 목록이 보이게 한 뒤 전체 목록 확인 준비 완료를 체크하고 전체 목록 확인을 누르면 후보 방 이름을 먼저 보여 줍니다. 앱이나 문자 인식이 막혀 있으면 화면에 보이는 앱 열기 또는 문자 인식 설정을 먼저 누릅니다.
   후보 이름이 맞는지 먼저 확인합니다.
   첫 화면에도 전체 목록 확인 -> 후보 확인 -> 목록 백업 실행 -> 결과 보기와 전체 저장 순서가 보입니다.
   번호가 붙은 후보 이름이 맞으면 아래 완료 카드의 목록 백업 실행을 누릅니다.
   전체 목록 확인 중 상한 늘려 전체 목록 다시 확인이 보이면 아직 방을 클릭하지 않은 상태이므로 그 버튼으로 후보 확인부터 다시 진행합니다.
   목록 백업 실행을 먼저 눌러도 바로 클릭을 시작하지 않고 "먼저 전체 목록 확인이 필요합니다" 카드가 나오며, 그 카드의 전체 목록 확인을 누르면 됩니다.
   목록 백업 실행은 왼쪽 목록을 끝까지 스크롤하고 새 방이 더 이상 나오지 않으면 자동으로 멈춥니다.
   완료 카드에 "왼쪽 목록 끝까지 백업했습니다"가 보이면 새 방이 더 이상 나오지 않아 목록 끝까지 확인한 상태입니다.
   카카오톡 목록 백업은 방마다 기본 120장까지 과거 대화를 읽고, 캡처 수 한도에 닿은 방은 결과 보기에서 이력 확인으로 표시됩니다.
   방 개수 상한 또는 페이지 상한에 도달했다는 안내가 보이면 전체 목록이 끝났다고 보장하기 어렵습니다. 전체 목록 확인 단계에서 더 올릴 수 없고 현재 후보만 저장해도 되면 같은 카드의 현재 후보로 목록 백업 실행을 누릅니다. 목록 백업 실행 뒤라면 저장된 결과는 먼저 볼 수 있고, 완료 카드의 상한 늘려 전체 목록 다시 확인을 누르면 더 큰 상한으로 전체 목록 확인부터 다시 시작합니다.
   목록 백업 실행 뒤 가능한 상한을 이미 최대로 올린 상태면 같은 버튼을 반복하지 않고 저장된 결과 보기와 확인 필요 방 보기로 빠진 방을 본 뒤 다시 백업합니다.
   끝나면 결과 보기에서 필요한 방은 표 파일 저장 또는 텍스트 저장으로 따로 저장하고, 왼쪽 목록 전체는 전체 표 파일 저장 또는 전체 텍스트 저장으로 한 번에 저장합니다.
   전체 저장은 현재 왼쪽 목록 기준입니다. 검색이나 필터를 바꾸면 저장 범위도 같이 바뀝니다.
   이 기능은 앱 화면을 읽어 순회하는 백업이고, 카카오톡이나 위챗 내부 DB를 직접 여는 방식은 아닙니다.

4. 문자 인식 준비가 필요하다고 나오면 중국어 문자 인식 설치 또는 한국어 문자 인식 설치를 누릅니다.
   백업 화면 위쪽의 3분 첫 백업 첫 버튼도 중국어 문자 인식 설치, 한국어 문자 인식 설치, 위챗 열기처럼 먼저 해야 할 일로 바뀝니다.
   Windows 권한 확인 창이 뜨면 예를 누릅니다. 창이 보이지 않으면 작업 표시줄의 새 권한 확인 창 또는 방패 아이콘을 확인합니다.
   자동 설치가 막히면 준비 확인의 Windows 언어 설정 열기를 누르고, 열린 Windows 설정에서 언어 추가를 누릅니다.
   카카오톡만 쓰면 한국어, 위챗을 쓰면 중국어(간체, 중국) 또는 중국어(번체, 대만/홍콩) 중 하나를 추가합니다.
   검색창에는 각각 한국어 또는 중국어를 입력합니다.
   설치할 기능을 묻는 화면이 나오면 기본 선택 그대로 설치합니다. 문자 인식 기능은 선택 해제하지 않습니다. Windows 표시 언어와 키보드는 바꾸지 않아도 됩니다.
   영어와 한국어만 설치된 PC는 위챗 백업 준비가 끝난 것으로 보지 않습니다.
   카카오톡과 위챗 중 한쪽만 준비된 상태는 실패가 아닙니다. 예를 들어 중국어 문자 인식이 아직 없어도 한국어 문자 인식과 카카오톡 앱이 준비되어 있으면 카카오톡 백업부터 진행할 수 있습니다.
   설치가 끝날 때까지 기다린 뒤 웹 화면으로 돌아와 상태 새로고침을 누릅니다. 바로 바뀌지 않으면 1~2분 뒤 한 번 더 누릅니다.

5. 끝나면 결과 보기를 누릅니다.
   결과 화면에서 정상은 읽을 수 있는 백업이고, 확인 필요는 실패가 아니라 발신자, 글자, 원본만 대조하면 된다는 뜻입니다.
   필요한 방을 선택한 뒤 표 파일 저장 또는 텍스트 저장을 누르면 지금 보고 있는 방만 따로 저장할 수 있습니다.
   결과가 비어 있으면 실패가 아니라 아직 이 화면에서 읽을 백업 파일이 없는 상태일 수 있습니다.
   결과 새로고침을 누릅니다.
   결과 보기 화면에서도 위챗 백업과 카카오톡 백업으로 돌아갈 수 있습니다.
   백업은 끝났는데 계속 비어 있으면 백업 화면의 같은 백업 다시 실행, 백업 폴더 열기, 진행 기록을 확인합니다.

기본 버튼으로 실행한 백업은 이 컴퓨터의 백업 폴더에 저장됩니다.
외부 검수/번역 기능은 기본으로 꺼져 있습니다.
처음 백업은 Windows 문자 인식으로 시작하고, 추가 검수는 필요할 때만 사용합니다.
무언가를 입력하라는 화면이 보여도 아무 값도 입력하지 않습니다.
예전 주소가 보여도 값을 찾지 않습니다. 화면이 자동으로 깨끗한 주소로 바뀝니다.
기본 더블클릭 사용에서는 값을 붙여 넣는 절차가 없습니다.
오래된 화면이나 접속 오류가 보이면 값을 찾지 말고 브라우저와 검은 창을 닫은 뒤 1_백업_시작.bat를 다시 실행합니다.
검은 창에 "백업 화면을 열지 못했습니다"가 보이면 같은 순서로 다시 열고, 계속 막히면 3_문제_확인.txt 맨 위의 빠른 해결 3단계를 먼저 봅니다.

scripts와 web 폴더는 백업 화면을 실행하는 데 필요한 프로그램 폴더입니다.
docs 폴더는 같은 안내를 보관한 참고 폴더입니다.
처음 사용하는 사람은 이 폴더들을 열 필요가 없습니다.
