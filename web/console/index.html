AI 보정은 선택 기능입니다. 처음 백업은 Windows 문자 인식으로 시작하고, 추가 검수는 필요할 때만 사용합니다. Windows 기본 백업은 앱 화면을 읽어 저장합니다.

앱 실행이나 문자 인식이 막혀 있으면 다음 화면에서 먼저 해야 할 버튼이 표시됩니다. 따로 입력할 내용은 없습니다.
