카카오톡/위챗 백업 문제 확인

대부분은 아래 3가지만 하면 해결됩니다.
1. 브라우저와 검은 창을 닫고 1_백업_시작.bat를 다시 더블클릭합니다.
2. 브라우저가 안 열리면 압축을 푼 폴더에 생기는 2_백업_화면.url을 더블클릭합니다.
3. 계속 막히면 4_실행도구_설치.url을 더블클릭해 설치한 뒤 1_백업_시작.bat를 다시 실행합니다.

처음 볼 것은 00_처음_여기부터.txt와 1_백업_시작.bat 두 개입니다.
Windows가 파일 확장자를 숨기면 1_백업_시작.bat가 1_백업_시작으로 보일 수 있습니다.
ZIP 파일 안에서 바로 실행하지 말고, 먼저 압축을 푼 뒤 실행합니다.
"필수 실행 파일이 빠졌습니다"가 보이면 압축이 제대로 풀리지 않았거나 일부 파일만 옮긴 상태입니다. 폴더 전체를 다시 압축 해제한 뒤 1_백업_시작.bat를 실행합니다.

아무 값도 입력하지 않습니다.
예전 주소 뒤에 붙은 긴 글자나 영어 오류 화면의 값을 찾거나 붙여 넣는 절차는 기본 사용에 없습니다.
오래된 화면이나 영어 접속 오류가 보이면 값을 입력하지 말고 1_백업_시작.bat를 다시 실행합니다.
백업 전 준비 상태만 보내야 하면 4_준비_보고서.bat를 실행합니다.
검증 보고서에 "요청 시간이 초과됐습니다"가 보이면 브라우저와 검은 창을 닫고 1_백업_시작.bat를 다시 실행합니다.

위챗 백업은 첫 화면, 왼쪽 메뉴, 결과 화면의 위챗 백업 버튼에서 들어갑니다.
여러 방을 한 번에 저장하려면 첫 화면의 위챗 통째 백업 확인 또는 카카오톡 통째 백업 확인을 누릅니다.
누른 뒤 앱 창을 앞에 두고 왼쪽 채팅 목록이 보이게 둡니다.
준비가 끝난 상태면 백업 화면의 전체 목록 확인 준비 완료 체크 위치로 이동합니다.
앱 창을 앞에 두고 왼쪽 채팅 목록이 보이게 한 뒤 전체 목록 확인 준비 완료를 체크하고 전체 목록 확인을 누르면 후보 방 이름을 먼저 보여 줍니다.
바로 클릭하지 않고 전체 목록 확인으로 후보 방 이름을 먼저 보여 준 뒤, 후보가 맞을 때 목록 백업 실행으로 이어집니다.
전체 목록 확인 중 상한 늘려 전체 목록 다시 확인이 보이면 아직 방을 클릭하지 않은 상태이므로 그 버튼을 먼저 누릅니다.
더 올릴 수 없는 상태에서 현재 후보만 저장해도 되면 같은 카드의 현재 후보로 목록 백업 실행을 누릅니다.
목록 백업 실행을 먼저 눌러도 바로 클릭을 시작하지 않고 "먼저 전체 목록 확인이 필요합니다" 카드가 나옵니다.

앱이 없으면 화면의 공식 설치 페이지 버튼을 누르고 설치와 로그인을 마친 뒤 상태 새로고침을 누릅니다.
문자 인식 준비가 필요하다고 나오면 중국어 문자 인식 설치 또는 한국어 문자 인식 설치를 누릅니다.
자동 설치가 막히면 Windows 언어 설정 열기에서 언어 추가를 누릅니다.
카카오톡만 쓰면 한국어, 위챗을 쓰면 중국어(간체, 중국) 또는 중국어(번체, 대만/홍콩) 중 하나를 기본 선택 그대로 설치합니다.

앱에서 백업할 방을 선택하고 앱 창을 앞에 둔 뒤 브라우저로 돌아와 방 선택 완료, 앱 창 앞에 둠 체크박스를 선택하고 백업을 누릅니다.
백업이 시작되면 끝날 때까지 카카오톡/위챗 창을 최소화하거나 다른 창으로 가리지 않습니다.

"백업 저장 폴더를 준비하지 못했습니다" 또는 저장할 수 없다는 안내가 나오면 압축을 푼 폴더를 바탕화면이나 문서처럼 내가 저장할 수 있는 일반 폴더로 옮긴 뒤 다시 실행합니다.
OneDrive 동기화 오류가 난 폴더, 회사 보안 폴더, 읽기 전용 폴더는 피합니다.

백업 화면을 실행하면 문제 확인 기록도 자동으로 최신 내용으로 갱신됩니다.
